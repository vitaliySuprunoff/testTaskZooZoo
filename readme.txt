zoo-zoo 1.0	24/07/2017

   Данная программа собирает объявления о животных с сайта zoo-zoo.ru, сохраняет их в базу данных
и предоставляет пользователю возможности по просмотру и поиску необходимого объявления

   Для корректной работы требуется:
- интернет соединение
- наличие на устройстве необходимой базы данных
Для создания БД требуется в MySQL Workbench выполнить sql скрипт, находящийся 
в корневой папке приложения, при помощи команды FILE-> RUN SQL SCRIPT.
Также обязательным условием является номер localhost'а - 3306. В противном слу-
чае необходимо внести изменения в переменную URL класса dbWorker.
